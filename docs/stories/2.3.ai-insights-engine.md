## Story

**As a** recruitment marketing specialist,
**I want** AI-generated insights that analyze my campaign data and surface actionable recommendations,
**so that** I can make faster, data-driven decisions without manually crunching numbers.

## Acceptance Criteria

1. AI insights section on the dashboard showing 3-5 key findings from the latest data
2. Uses Claude API to analyze joined metrics data and produce natural language insights
3. Insight categories: top/bottom performers, anomalies, trends, budget recommendations
4. Each insight includes: finding, supporting data, recommended action
5. "Refresh insights" button to re-analyze with latest data
6. Insights are cached and timestamped (don't re-call API on every page load)
7. Handles API errors gracefully (show cached insights or "unable to generate" message)
8. Prompt engineering: system prompt includes Indeed Flex business context (funnel stages, terminology, benchmarks)
