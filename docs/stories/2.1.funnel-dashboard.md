## Story

**As a** recruitment marketing specialist,
**I want** a dashboard showing the 7-stage onboarding funnel by client and market,
**so that** I can see where candidates drop off and which client/market combinations perform best.

## Acceptance Criteria

1. Dashboard page at `/dashboard` with funnel visualization
2. Funnel chart showing 7 stages: Accounts Created → Role Verified → OB Task → Platform Verified → Ready to Book → 1st Shift Booked → 1st Shift Completed
3. Filter by client, market, and date range
4. Summary cards at top: total accounts created, total verified, total shifts completed, overall CR%, total spend
5. Client performance table: rows per client showing Created, Verified, Platform V, Booked, Completed, CR%
6. Market performance table: same metrics grouped by market
7. Stage-to-stage conversion rates displayed between funnel stages
8. Color coding: green (>20% CR), yellow (10-20%), red (<10%) for overall conversion
9. Data sourced from `joined_metrics` table (populated by Story 1.3)
10. Empty state with guidance when no data uploaded yet
