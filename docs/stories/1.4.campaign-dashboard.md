## Story

**As a** marketing team member,
**I want** a dashboard that lists all active campaigns across channels with key metrics,
**so that** I can see the overall state of Indeed Flex's recruitment advertising at a glance.

## Acceptance Criteria

1. Dashboard page at `/dashboard` displays a table/list of all campaigns across connected channels
2. Each campaign row shows: campaign name, channel (with icon), status, spend, clicks, applications, CPA, CTR
3. Campaigns are sortable by any column
4. Campaigns are filterable by channel, status (active/paused/ended), and date range
5. Dashboard loads within 3 seconds for up to 500 campaigns (NFR1)
6. Channel status indicators show connected/disconnected state per channel
7. Summary cards at top show: total spend, total applications, blended CPA, active campaign count
8. Empty state shown gracefully when no campaigns exist or channels not connected
9. Dashboard auto-refreshes data every 15 minutes (NFR2)
10. Responsive layout works on desktop and tablet (NFR6)

## Dev Notes

### Architecture
- Page: `src/app/dashboard/page.tsx`
- Components: `src/components/dashboard/` (SummaryCards, CampaignTable, ChannelStatus)
- API: `src/app/api/campaigns/route.ts` — aggregates from all channel services
- Data fetching: SWR or React Query with 15min revalidation
- Depends on: Story 1.1, 1.2 (auth), 1.3 (channel APIs)

### UI Notes
- Use Tailwind CSS for styling
- Table should use virtualization if >100 rows for performance
- Channel icons: color-coded badges (Google=blue, Meta=blue, Indeed=purple, Bing=teal, Reddit=orange, CL=purple)
- Mobile: Stack summary cards, horizontal scroll on table

### Testing
- Test location: `tests/components/dashboard/`
- Use React Testing Library for component tests
- Mock channel API responses for table data
