## Story

**As a** PPC specialist,
**I want** an AI-powered ad copy generator that creates recruitment ad variants per channel with proper character limits,
**so that** I can quickly produce high-quality ad copy without manual writing for every campaign.

## Acceptance Criteria

1. Ad copy generator page at `/tools/ad-copy-generator`
2. Input form: job title, location, pay range (min/max), shift type, channel selection
3. Generates ad copy variants for selected channel(s) using template patterns and AI enhancement
4. Each variant displays character count and warns if over channel limit (Google headlines=30, descriptions=90; Meta primary=125, headline=40)
5. Support for all channels: Google Search, Meta, Reddit, Indeed (sponsored title)
6. "Generate All Channels" button produces copy for all channels simultaneously
7. Copy-to-clipboard per variant and bulk export as CSV
8. Validate button checks all copy against channel character limits and flags violations
9. Integration with campaign creation wizard (Story 2.1) creative step
