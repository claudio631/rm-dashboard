## Story

**As a** PPC specialist,
**I want** a guided campaign creation workflow that walks me through objective, targeting, budget, and creative setup for any channel,
**so that** I can launch new recruitment campaigns quickly and consistently.

## Acceptance Criteria

1. Campaign creation page at `/campaigns/new` with multi-step wizard (Objective → Targeting → Budget → Creative → Review → Launch)
2. Channel selection as first step with supported channels displayed (Indeed, Google, Meta, Bing, Reddit, Craigslist)
3. Objective selection: candidate acquisition, brand awareness, retargeting
4. Targeting configuration: geographic markets, job categories, audience segments (from audience builder)
5. Budget configuration: daily budget, total budget, campaign duration, bid strategy recommendation per channel
6. Creative step: ad copy input with character limit validation per channel, option to use ad copy generator
7. Review step: full campaign summary before launch with UTM parameters auto-generated
8. Campaign saved to database with status "draft" or "active"
9. Campaign naming follows convention: `{channel}-{market}-{category}-{objective}-{date}`
10. Form state persists if user navigates away and returns (draft saving)
