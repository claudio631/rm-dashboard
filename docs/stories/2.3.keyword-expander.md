## Story

**As a** PPC specialist,
**I want** a keyword expansion tool that generates location + job type + modifier keyword combinations from seed lists,
**so that** I can quickly build comprehensive keyword sets for recruitment search campaigns.

## Acceptance Criteria

1. Keyword expander page at `/tools/keyword-expander`
2. Input: select job category (warehouse, delivery, retail, etc.) or enter custom seed keywords
3. Input: target locations (comma-separated cities/states)
4. Output: expanded keywords in three match types: exact, phrase, broad
5. Built-in seed keyword lists for all Indeed Flex job categories (8 categories from squad data)
6. Modifier expansion includes: "near me", "hiring now", "no experience", "weekly pay", "flexible hours", shift types
7. Negative keyword suggestions generated automatically
8. Export as CSV or copy formatted for Google Ads / Bing Ads import
9. Display total keyword count and breakdown by match type
10. Option to include/exclude modifiers and shift types via toggles
