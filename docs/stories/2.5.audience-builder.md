## Story

**As a** PPC specialist,
**I want** to build and manage candidate audience segments from pre-built templates and custom definitions,
**so that** I can target the right candidates across channels with consistent audience strategies.

## Acceptance Criteria

1. Audience builder page at `/tools/audience-builder`
2. Pre-built audience templates available: warehouse seekers, delivery drivers, flex work seekers, students/seasonal, retargeting (past applicants), lookalike (top workers)
3. Create custom audience with: name, demographics (age, gender), interests, behaviors, exclusions, locations
4. Each audience shows channel-specific targeting recommendations (Google audiences, Meta interests, Reddit subreddits)
5. Combine multiple audiences with OR/AND logic
6. Save audiences to database for reuse across campaigns
7. Integration with campaign creation wizard targeting step (Story 2.1)
8. Export audience definition as formatted spec per channel
