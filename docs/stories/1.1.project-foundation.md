## Story

**As a** recruitment marketing specialist,
**I want** a working application foundation with file upload capability,
**so that** I have a base platform to drop my daily report files into.

## Acceptance Criteria

1. Next.js 14+ project with App Router, TypeScript strict mode, Tailwind CSS, and shadcn/ui configured
2. Landing page at `/` with app name "RM Team AI" and navigation to Dashboard and Tools sections
3. `npm run dev`, `npm run build`, `npm run lint`, `npm run typecheck`, `npm test` all pass
4. SQLite database configured with Drizzle ORM (or Prisma) for local development
5. Database schema includes: `uploads` table (id, filename, file_type, uploaded_at, row_count, status)
6. Environment variable setup with `.env.example` containing `CLAUDE_API_KEY` and `SLACK_WEBHOOK_URL` placeholders
7. Project folder structure: `src/app/`, `src/lib/`, `src/components/`, `src/services/`, `src/types/`, `src/data/`
8. Basic smoke test verifying the landing page renders
