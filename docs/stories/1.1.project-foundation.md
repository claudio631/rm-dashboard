## Story

**As a** developer,
**I want** a fully configured project foundation with package.json, folder structure, linting, TypeScript, and a running dev server,
**so that** all subsequent stories have a solid, consistent base to build upon.

## Acceptance Criteria

1. Project initialized with `package.json` including name `ppc-manager-ai`, version `0.1.0`
2. TypeScript configured with strict mode enabled (`tsconfig.json`)
3. ESLint and Prettier configured with consistent rules
4. Folder structure created: `src/`, `src/lib/`, `src/components/`, `src/services/`, `src/types/`, `src/config/`, `tests/`
5. Next.js (App Router) or equivalent framework installed and running on `localhost:3000`
6. Tailwind CSS configured with Indeed Flex brand-compatible color tokens
7. Environment variable handling set up (`.env.example` with placeholder keys for channel APIs)
8. `npm run dev`, `npm run build`, `npm run lint`, `npm run typecheck`, `npm test` all execute successfully
9. Basic health check page renders at `/` confirming the app is running
10. Git pre-commit hook runs lint on staged files

## Dev Notes

### Architecture
- Framework: Next.js 14+ with App Router (React Server Components)
- Styling: Tailwind CSS
- Language: TypeScript (strict mode)
- Package manager: npm
- This is the foundational story — all subsequent stories depend on this

### Testing
- Test framework: Jest + React Testing Library
- Test location: `tests/` directory mirroring `src/` structure
- Minimum: Smoke tests for page rendering and build validation
- Run with: `npm test`
