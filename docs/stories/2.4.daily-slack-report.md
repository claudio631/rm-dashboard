## Story

**As a** recruitment marketing specialist,
**I want** to generate the daily EOD Slack report with one click instead of manually compiling it with Gemini,
**so that** I save 30+ minutes daily and the report is consistent every time.

## Acceptance Criteria

1. "Generate Slack Report" button on the dashboard
2. Report follows the exact current format: Key client unique accounts (client → location → Created → Verified (+N daily delta)), Indeed Spend Comparison (month-to-date), Open Campaigns list
3. Daily delta (+N) calculated by comparing today's data to the previous upload's data
4. Report preview shown in-app before posting (editable text area)
5. "Copy to Clipboard" button to paste into Slack manually
6. "Post to Slack" button that sends via Slack Incoming Webhook
7. Report includes AI-generated summary (1-2 sentences) at the top highlighting key changes
8. Report generation works with partial data (e.g., only FHS + Tableau uploaded, no Indeed spend yet)
9. Report history: previous reports stored with timestamp for reference
