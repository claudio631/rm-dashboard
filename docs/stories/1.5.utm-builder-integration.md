## Story

**As a** marketing team member,
**I want** an integrated UTM builder that automatically generates tracking parameters following our naming convention,
**so that** all campaign links have consistent, reliable attribution tracking.

## Acceptance Criteria

1. UTM builder page at `/tools/utm-builder` with form inputs for source, medium, market, category, objective, date, content, term
2. Real-time preview of generated UTM string and full tracked URL as user types
3. Validation enforces naming convention: lowercase, hyphens only, no special characters
4. Dropdown suggestions for source (pre-populated with Indeed Flex channels) and medium (cpc, cpm, social, email, etc.)
5. Batch generation mode: generate UTMs for multiple ad variants with different `utm_content` values
6. Copy-to-clipboard button for generated URLs
7. History of recently generated UTMs stored in browser (last 20)
8. Integration with campaign creation flow (Story 2.1) via shared UTM utility
9. UTM parsing tool: paste a URL and extract/display its UTM parameters
