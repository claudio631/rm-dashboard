# Campaign Launch Workflow

## Overview

Coordinates all 5 squad specialists to launch a synchronized multi-channel recruitment campaign for Indeed Flex.

## Phases

### Phase 1: Foundation (Day -7 to -5)
**Lead:** @analytics-performance-lead (Metric)
- Define campaign KPIs and success criteria
- Set up UTM taxonomy for the campaign
- Configure conversion tracking across platforms
- Prepare reporting dashboard

### Phase 2: Content & SEO (Day -5 to -3)
**Lead:** @seo-content-strategist (Scout)
- Optimize target job postings
- Prepare landing pages
- Validate Google for Jobs schema
- Create supporting organic content

### Phase 3: Campaign Build (Day -3 to -1)
**Lead:** @ppc-paid-media-specialist (Parker)
- Build campaigns in each platform
- Set targeting and audiences
- Upload creatives and ad copy
- Configure bid strategies and budgets

### Phase 4: Automation Setup (Day -2 to -1)
**Lead:** @ai-automation-specialist (Synth)
- Enable smart bidding strategies
- Set up rule-based automations
- Configure performance alerts
- Test all integrations

### Phase 5: Nurture Activation (Day -1)
**Lead:** @crm-email-specialist (Relay)
- Activate welcome sequences for new channel
- Prepare re-engagement triggers
- Test email/SMS delivery
- Verify personalization tokens

### Phase 6: Launch (Day 0)
**Lead:** @ppc-paid-media-specialist (Parker)
- Activate all campaigns simultaneously
- Verify ads serving correctly across channels
- Monitor initial spend pacing
- Address any disapprovals or errors

### Phase 7: Post-Launch (Day +1 to +2)
**Lead:** @analytics-performance-lead (Metric)
- 24-hour performance check
- 48-hour comprehensive report
- Initial optimization recommendations
- Escalate any underperforming channels
