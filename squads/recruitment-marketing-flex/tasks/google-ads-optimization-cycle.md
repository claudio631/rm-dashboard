# Task: Google Ads Weekly Optimization Cycle

> Agent: @ppc-paid-media-specialist
> Elicit: false
> Dependencies: google-ads-optimization-checklist.md, budget-optimization-guide.md

## Objective
Execute the weekly optimization routine for all active Google Ads campaigns to maintain and improve performance.

## Pre-Conditions
- Active Google Ads campaigns running for 2+ weeks
- Conversion tracking verified
- Previous week's data available

## Frequency
**Weekly** — every Monday (or first business day of the week)

## Workflow

### Step 1: Budget Pacing Check (15 min)
- [ ] Check month-to-date spend vs. monthly budget
- [ ] Calculate pacing: (Spend ÷ Days Elapsed) × Days in Month
- [ ] If underspending >10%: check impression share, consider bid increases
- [ ] If overspending >10%: add negatives, lower bids, tighten targeting
- [ ] Reallocate budget from capped campaigns to those with headroom
- [ ] Document any budget changes

### Step 2: Search Terms Review (20 min)
- [ ] Pull Search Terms Report for last 7 days
- [ ] Identify irrelevant search terms → add as negative keywords
- [ ] Identify high-performing new terms → add as keywords
- [ ] Check for brand terms being triggered (if unwanted)
- [ ] Review negative keyword lists for over-blocking
- [ ] Log all additions and removals

### Step 3: Bid & Bidding Strategy Check (15 min)
- [ ] Review Smart Bidding performance:
  - Target CPA: actual CPA vs. target
  - Target ROAS: actual ROAS vs. target
  - Maximize Conversions: conversion volume trend
- [ ] Adjust targets if needed (small increments only, ±10-15%)
- [ ] Check if campaigns are limited by budget
- [ ] Review device bid adjustments
- [ ] Review location bid adjustments

### Step 4: Ad Performance Review (15 min)
- [ ] Check ad-level CTR and conversion rates
- [ ] Identify ads with below-average performance
- [ ] Check Ad Strength indicators
- [ ] Review asset performance labels (PMax)
- [ ] Plan ad refreshes if assets have "Low" label
- [ ] Note winning themes for future creative

### Step 5: Quality Score Check (10 min)
- [ ] Review keywords with QS < 5 → investigate and fix
- [ ] Check factors: Expected CTR, Ad Relevance, Landing Page Experience
- [ ] For low Landing Page Experience: flag for web team
- [ ] For low Ad Relevance: align ad copy with keywords
- [ ] For low Expected CTR: improve ad copy or pause keyword

### Step 6: Conversion Health Check (10 min)
- [ ] Verify conversion tags are firing (no sudden drops)
- [ ] Check conversion rate trends (alert if >20% change)
- [ ] Review conversion by type (application, registration, call)
- [ ] Verify conversion values are tracking correctly
- [ ] Flag any tracking issues for immediate resolution

### Step 7: Competitive Landscape (10 min)
- [ ] Review Auction Insights for Search campaigns
- [ ] Note any significant changes in impression share
- [ ] Identify new competitors
- [ ] Adjust strategy if competitor activity changed materially

### Step 8: Weekly Summary (15 min)
- [ ] Document all changes made this week
- [ ] Record key metrics: spend, conversions, CPA, ROAS
- [ ] List top 3 wins and top 3 concerns
- [ ] Set focus areas for next week
- [ ] Flag anything needing stakeholder attention

## Time Estimate
~110 minutes (2 hours) per week for standard account

## Post-Conditions
- All campaigns reviewed and optimized
- Negative keywords updated
- Bids and budgets adjusted
- Weekly summary documented
- Issues flagged and escalated

## Outputs
- Weekly optimization log
- Updated negative keyword lists
- Bid/budget change log
- Escalation items (if any)
