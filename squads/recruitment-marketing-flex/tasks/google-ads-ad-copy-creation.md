# Task: Google Ads Ad Copy Creation

> Agent: @ppc-paid-media-specialist
> Elicit: true
> Dependencies: ad-copywriting-playbook.md

## Objective
Create high-performing ad copy for Google Ads campaigns following Google's copywriting best practices and recruitment marketing context.

## Pre-Conditions
- Keywords and ad groups defined
- Landing pages identified per ad group
- Brand guidelines available
- Unique value propositions defined

## Workflow

### Step 1: Brief Collection
**Elicit:** What are the campaign details?
- [ ] Job title/category being advertised
- [ ] Target market/location
- [ ] Key benefits to highlight (pay, flexibility, speed)
- [ ] Unique selling points vs. competitors
- [ ] Specific CTA goal (apply, register, call)
- [ ] Any compliance/legal requirements for ad text

### Step 2: Headline Generation
Create 15 headlines (30 characters max each) per ad group:

**Category A: Job + Location (3-4 headlines)**
- "{Job Title} Jobs — {City}"
- "{Job Title} — Hiring in {City}"
- "{City} {Job Title} Positions"

**Category B: Pay + Benefits (3-4 headlines)**
- "${Pay}/hr — Start This Week"
- "Weekly Pay + Flexible Hours"
- "Same-Day Pay Available"

**Category C: Urgency + Action (3-4 headlines)**
- "We're Hiring — Apply Now"
- "Start As Soon As Tomorrow"
- "{Number} Open Positions"

**Category D: Value Proposition (3-4 headlines)**
- "No Experience Needed"
- "Choose Your Own Schedule"
- "Paid Training Provided"

**Category E: Question / Engagement (2-3 headlines)**
- "Looking for {Job Type} Work?"
- "Need Flexible Hours?"

### Step 3: Description Writing
Create 4 descriptions (90 characters max each) per ad group:

**Description 1: Primary value proposition + CTA**
```
Find {job type} jobs near you. ${pay}/hr, flexible shifts, weekly pay. Apply online in under 2 minutes.
```

**Description 2: Benefits-focused**
```
No experience needed. Paid training, same-day pay, and the freedom to choose your schedule. Sign up today.
```

**Description 3: Urgency-focused**
```
{Number} open positions in {city}. Start working as soon as this week. Apply now — spots are filling fast.
```

**Description 4: Trust-focused**
```
Join {number} workers who choose flexible shifts with us. Rated {rating} on {platform}. Get started free.
```

### Step 4: Display URL Paths
Set 2 display URL paths (15 chars each):
- Path 1: Job category (e.g., "Warehouse-Jobs")
- Path 2: Location or CTA (e.g., "Houston" or "Apply-Now")

### Step 5: Ad Extensions
- [ ] Write 4+ sitelinks with descriptions:
  - "Browse All Jobs" → job listing page
  - "See Pay Rates" → compensation page
  - "How It Works" → process page
  - "Apply Now" → application page
- [ ] Write 4+ callout extensions:
  - "Same-Day Pay"
  - "Flexible Schedules"
  - "No Experience Needed"
  - "Weekly Paychecks"
- [ ] Write structured snippets:
  - Header: "Types" → "Warehouse, Delivery, Caregiver, Retail, Events"
  - Header: "Neighborhoods" → location list

### Step 6: Ad Strength Check
- [ ] Enter all assets in Google Ads
- [ ] Check Ad Strength indicator
- [ ] Target: "Good" or "Excellent"
- [ ] If below target, add more headline variety and keyword inclusion
- [ ] Ensure at least 1 keyword appears in headlines and descriptions

### Step 7: A/B Test Plan
- [ ] Identify one variable to test first:
  - Pay rate in headline vs. not
  - Urgency CTA vs. benefit CTA
  - Question headline vs. statement
  - Specific numbers vs. general claims
- [ ] Set test duration (minimum 2 weeks)
- [ ] Define success metric (CTR, conversion rate, or CPA)
- [ ] Document test hypothesis

## Post-Conditions
- 15 headlines + 4 descriptions per ad group
- Ad extensions configured
- Ad Strength at "Good" or better
- A/B test plan documented

## Outputs
- Ad copy spreadsheet (all headlines, descriptions, extensions)
- A/B test plan document
- Ad Strength screenshots
