# Recruitment Marketing Flex Squad

Multi-channel recruitment marketing specialists for **Indeed Flex** staffing agency.

## Squad Overview

This squad manages end-to-end recruitment marketing across paid, organic, CRM, and AI-driven channels to acquire flexible/temporary workers for Indeed Flex.

## Agents

| Agent | Name | Role | Activation |
|-------|------|------|------------|
| 💰 PPC/Paid Media | Parker | Manages paid campaigns across all channels | `@ppc-paid-media-specialist` |
| 🔍 SEO/Content | Scout | Organic search, job posting SEO, content strategy | `@seo-content-strategist` |
| 📧 CRM/Email | Relay | Candidate nurturing, email/SMS, CRM management | `@crm-email-specialist` |
| 📊 Analytics | Metric | Reporting, attribution, cost-per-hire optimization | `@analytics-performance-lead` |

## Channels

**Primary:** Indeed Ads, Google Ads, Meta Ads (Facebook & Instagram)
**Secondary:** Bing Ads, Reddit Ads, Craigslist, LinkedIn Ads
**Programmatic:** Appcast, Joveo, PandoLogic

## Quick Start

1. Activate an agent: `@ppc-paid-media-specialist` (or any agent above)
2. Run a task: `*create-campaign`, `*audit-seo`, `*weekly-report`, etc.
3. Launch a campaign: Use the `campaign-launch-workflow` for coordinated launches
4. Validate: Use `campaign-launch-checklist` before going live

## North Star Metrics

- **Blended Cost-per-Hire** — Total marketing spend / total hires
- **Fill Rate** — Positions filled / positions posted
- **Time-to-Fill** — Days from posted to hired
- **Candidate Quality** — 90-day retention rate by source
