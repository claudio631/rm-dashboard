# Google Ads Setup Checklist

> Source: [Google Business Resource Hub](https://business.google.com/us/resources/)
> Use before launching any new Google Ads campaign

## Account Setup
- [ ] Google Ads account created and verified
- [ ] Billing information configured
- [ ] Correct time zone and currency set
- [ ] Google Analytics 4 linked
- [ ] Google Tag installed sitewide
- [ ] Enhanced conversions enabled
- [ ] Auto-tagging enabled for GA4

## Conversion Tracking
- [ ] Primary conversion action created (e.g., application submitted)
- [ ] Secondary conversion actions created (e.g., registration started)
- [ ] Conversion values assigned
- [ ] Conversion windows set (30-90 days click-through)
- [ ] Event snippets verified on thank-you/confirmation pages
- [ ] Test conversions firing correctly (use Google Tag Assistant)
- [ ] Attribution model set (data-driven recommended)

## Campaign Configuration
- [ ] Campaign named using convention: `{Channel}-{JobCategory}-{Market}-{Goal}`
- [ ] Campaign type selected (Search, PMax, Display, Video)
- [ ] Campaign goal aligned with conversion actions
- [ ] Network settings correct (Search only vs. Search + Partners)
- [ ] Geographic targeting set to markets with active positions
- [ ] Language targeting set correctly
- [ ] Ad schedule configured (candidate peak hours)
- [ ] Start and end dates set (if applicable)

## Budget & Bidding
- [ ] Daily budget set (monthly ÷ 30.4)
- [ ] Bidding strategy selected and configured
- [ ] Bid limits set (if using Target CPA/ROAS)
- [ ] Budget is ≥10x target CPA (for Smart Bidding)
- [ ] Monthly budget ceiling documented

## Keywords (Search Campaigns)
- [ ] Keywords researched using Keyword Planner
- [ ] Keywords organized into themed ad groups
- [ ] Match types set (broad match for Smart Bidding)
- [ ] Negative keyword list created and applied
- [ ] No duplicate keywords across ad groups
- [ ] Each ad group maps to a specific landing page

## Ads
- [ ] Responsive Search Ads created (min 1 per ad group)
- [ ] 8-15 headlines written (30 chars each)
- [ ] 3-4 descriptions written (90 chars each)
- [ ] Display URL paths set
- [ ] At least 1 keyword included in headlines
- [ ] Clear call-to-action in ads
- [ ] Ad Strength indicator: "Good" or "Excellent"
- [ ] Ads proofread for grammar and spelling
- [ ] No ALL CAPS or excessive punctuation
- [ ] Ads comply with Google Ads policies

## Assets (Performance Max)
- [ ] 15 headlines uploaded
- [ ] 5 long headlines uploaded
- [ ] 5 descriptions uploaded
- [ ] 3+ landscape images (1200x628)
- [ ] 3+ square images (1200x1200)
- [ ] 1+ portrait images (960x1200)
- [ ] Logo uploaded (square + landscape)
- [ ] 1+ video uploaded (or auto-generation accepted)
- [ ] Audience signals configured
- [ ] Asset group strength: "Excellent"

## Ad Extensions
- [ ] Sitelinks added (4+ with descriptions)
- [ ] Callout extensions added (4+)
- [ ] Structured snippets added
- [ ] Location extension enabled (if applicable)
- [ ] Call extension enabled (if phone applications accepted)
- [ ] Lead form extension (if using in-ad forms)
- [ ] Image extensions uploaded

## Landing Pages
- [ ] Landing page matches ad promise (relevance)
- [ ] Mobile-responsive design verified
- [ ] Page loads in <3 seconds (test with PageSpeed Insights)
- [ ] Clear CTA above the fold
- [ ] Application form is short and mobile-friendly
- [ ] No intrusive pop-ups on mobile
- [ ] SSL certificate active (HTTPS)

## Final Pre-Launch
- [ ] All settings reviewed in campaign preview
- [ ] Budget confirmed with stakeholders
- [ ] Monitoring schedule set (daily for first 2 weeks)
- [ ] 2-week no-change period acknowledged (learning phase)
- [ ] Emergency contacts documented (who to notify if issues)
- [ ] Campaign launched
- [ ] First impressions and clicks verified within 2 hours
