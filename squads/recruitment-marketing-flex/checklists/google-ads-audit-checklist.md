# Google Ads Full Account Audit Checklist

> Source: [Google Business Resource Hub](https://business.google.com/us/resources/)
> Run quarterly or when onboarding a new account

## 1. Account Structure
- [ ] Account hierarchy is logical (campaigns → ad groups → ads → keywords)
- [ ] Campaign naming convention is consistent and descriptive
- [ ] Campaigns organized by job category, market, or goal
- [ ] Ad groups have tight keyword themes (≤20 keywords each)
- [ ] No overlapping keywords across ad groups
- [ ] Shared budgets assessed (vs. individual campaign budgets)
- [ ] Account structure mirrors landing page / website structure

## 2. Campaign Settings
- [ ] All campaigns have correct geographic targeting
- [ ] No campaigns targeting "people interested in" (vs. "people in" your locations)
- [ ] Language targeting appropriate for each market
- [ ] Ad schedule settings reviewed (not missing peak hours)
- [ ] Network settings correct (Search vs. Display separation)
- [ ] Campaign goals aligned with business objectives
- [ ] Start/end dates appropriate (no expired campaigns still active)

## 3. Conversion Tracking
- [ ] All conversion actions listed and categorized
- [ ] Primary vs. secondary conversions properly designated
- [ ] Conversion values assigned and accurate
- [ ] Google Tag verified on all pages (Tag Assistant)
- [ ] Enhanced conversions enabled
- [ ] Conversion windows appropriate (not too short/long)
- [ ] Attribution model reviewed (data-driven recommended)
- [ ] GA4 linked and importing conversions
- [ ] No duplicate conversion counting
- [ ] Offline conversion import configured (if applicable)

## 4. Budget & Bidding
- [ ] Monthly budget aligned with business goals
- [ ] Daily budgets distributed appropriately across campaigns
- [ ] No campaigns consistently limited by budget (unless intentional)
- [ ] Bidding strategies appropriate for each campaign's maturity:
  - New: Maximize Conversions
  - Established: Target CPA or Target ROAS
- [ ] Bid limits reviewed (not unnecessarily restrictive)
- [ ] Smart Bidding campaigns have sufficient conversion data (15+/30 days)
- [ ] No manual bidding on campaigns that would benefit from automation

## 5. Keywords (Search)
- [ ] Keyword list is comprehensive for target job categories
- [ ] Match types aligned with bidding strategy (broad for Smart Bidding)
- [ ] No keywords with 0 impressions for 30+ days (paused or removed)
- [ ] Quality Scores reviewed (average ≥6, target ≥7)
- [ ] Low QS keywords investigated and optimized or paused
- [ ] Search Terms Report reviewed (last 90 days)
- [ ] Negative keyword lists up to date and comprehensive
- [ ] No conflicting negatives (blocking good keywords)
- [ ] Long-tail opportunities identified
- [ ] Competitor keyword strategy assessed

## 6. Ads & Creative
- [ ] All ad groups have at least 1 active Responsive Search Ad
- [ ] RSAs have 8-15 headlines and 3-4 descriptions
- [ ] Ad Strength at "Good" or "Excellent" for all ads
- [ ] Headlines include relevant keywords
- [ ] Clear CTAs in every ad
- [ ] No policy violations or disapproved ads
- [ ] No ads with "Poor" or "Average" Ad Strength (optimize)
- [ ] Ad copy tested and refreshed within last 90 days
- [ ] Messaging consistent with landing page content
- [ ] PMax asset groups have sufficient diverse assets

## 7. Ad Extensions
- [ ] Sitelinks active (4+ per campaign)
- [ ] Callout extensions active (4+ per campaign)
- [ ] Structured snippets active
- [ ] Location extensions enabled (if applicable)
- [ ] Call extensions enabled (if phone applications)
- [ ] Lead form extensions reviewed (if used)
- [ ] Image extensions uploaded
- [ ] No disapproved extensions
- [ ] Extension performance reviewed (remove low performers)

## 8. Audiences
- [ ] Remarketing lists created and active
- [ ] Customer Match lists uploaded and current
- [ ] Audience segments reviewed for performance
- [ ] In-market and affinity audiences tested
- [ ] Audience exclusions set (existing converters)
- [ ] Audience signals configured for PMax campaigns
- [ ] Similar audiences/lookalikes utilized
- [ ] Demographics reviewed and adjusted

## 9. Landing Pages
- [ ] All ad URLs resolve correctly (no 404s)
- [ ] Landing pages match ad messaging
- [ ] Mobile experience tested on actual devices
- [ ] Page speed: <3 seconds on mobile (PageSpeed Insights)
- [ ] Core Web Vitals passing (LCP, FID, CLS)
- [ ] Forms functional and short (3-5 fields)
- [ ] SSL active (HTTPS)
- [ ] No intrusive pop-ups on mobile
- [ ] Clear CTA above the fold
- [ ] Multiple landing page variants tested

## 10. Performance Metrics
- [ ] CPA within target range
- [ ] Conversion rate acceptable (>3% Search, >1% Display)
- [ ] CTR healthy (>3% Search, >0.5% Display)
- [ ] Impression share acceptable (>60% Search)
- [ ] ROAS positive (if tracked)
- [ ] Quality Score average ≥6
- [ ] Wasted spend identified and addressed
- [ ] Top performers identified for scaling
- [ ] Bottom performers identified for optimization or pausing
- [ ] Month-over-month trends documented

## 11. Competitive Analysis
- [ ] Auction Insights reviewed
- [ ] Competitor overlap and outranking assessed
- [ ] Competitive share of voice evaluated
- [ ] Ad copy differentiation from competitors verified
- [ ] Competitive gaps identified (opportunities)

## 12. Compliance & Governance
- [ ] All ads compliant with Google Ads policies
- [ ] Employment ad policies followed (fair hiring, non-discrimination)
- [ ] Billing and payment methods current
- [ ] User access/permissions reviewed (remove old users)
- [ ] Change history reviewed for unauthorized changes
- [ ] Automated rules reviewed (if any)
- [ ] Scripts reviewed (if any)
