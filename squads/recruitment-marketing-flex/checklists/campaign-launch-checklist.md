# Campaign Launch Checklist

Pre-launch validation for Indeed Flex multi-channel recruitment campaigns.

## Tracking & Analytics
- [ ] UTM parameters defined and documented
- [ ] Google Analytics 4 events configured
- [ ] Conversion tracking pixels installed on all landing pages
- [ ] Offline conversion import set up (apply → hire)
- [ ] Reporting dashboard ready with all metrics
- [ ] Attribution model selected and configured

## Landing Pages & SEO
- [ ] Landing pages live and mobile-responsive
- [ ] Page load speed <2.5s (Core Web Vitals pass)
- [ ] Google for Jobs schema markup validated
- [ ] Apply CTA visible above the fold
- [ ] Job descriptions optimized for target keywords
- [ ] Canonical tags set correctly

## Paid Campaigns
- [ ] Campaigns built in all target platforms
- [ ] Ad copy reviewed and approved
- [ ] Targeting parameters verified (geo, demo, interests)
- [ ] Negative keywords/exclusions added
- [ ] Bid strategy set per channel
- [ ] Daily budgets configured with pacing rules
- [ ] Ad extensions/assets added (Google/Bing)
- [ ] Campaign names follow naming convention

## CRM & Nurturing
- [ ] Welcome sequence activated for new applicants
- [ ] Job match notifications configured
- [ ] Unsubscribe links functional
- [ ] Personalization tokens rendering correctly
- [ ] SMS opt-in flow compliant
- [ ] CRM segments created for campaign audience

## Automation
- [ ] Smart bidding enabled where applicable
- [ ] Performance alert thresholds set
- [ ] Budget pacing rules active
- [ ] Automated reports scheduled
- [ ] Fallback rules in place for automation failures

## Compliance
- [ ] Job ads compliant with local labor regulations
- [ ] Equal opportunity language included
- [ ] Privacy policy link on all landing pages
- [ ] Data collection consent where required (GDPR/CCPA)
- [ ] CAN-SPAM compliance for all emails

## Final Sign-off
- [ ] All squad members confirmed ready
- [ ] Stakeholder approval received
- [ ] Launch date/time confirmed
- [ ] War room / monitoring plan in place for first 48 hours
