# Google Ads Weekly Optimization Checklist

> Source: [Google Business Resource Hub](https://business.google.com/us/resources/)
> Run weekly on Monday (or first business day)

## Budget & Pacing
- [ ] Month-to-date spend reviewed vs. monthly budget
- [ ] Pacing calculated: (Spend ÷ Days Elapsed) × Days in Month
- [ ] Underspending campaigns investigated (impression share, bid levels)
- [ ] Overspending campaigns adjusted (negatives, bids, targeting)
- [ ] Budget reallocated from low performers to high performers
- [ ] Any budget changes documented

## Search Terms & Keywords
- [ ] Search Terms Report reviewed (last 7 days)
- [ ] Irrelevant terms added as negative keywords
- [ ] High-performing new terms added as keywords
- [ ] Negative keyword lists checked for over-blocking
- [ ] Keyword performance reviewed (pause keywords with high spend, zero conversions after 2+ weeks)

## Bids & Bidding Strategy
- [ ] Smart Bidding actual vs. target CPA/ROAS reviewed
- [ ] Bid adjustments made if targets are off by >15%
- [ ] "Limited by budget" warnings addressed
- [ ] Device bid adjustments reviewed
- [ ] Location bid adjustments reviewed
- [ ] Schedule bid adjustments reviewed

## Ad Performance
- [ ] Ad-level CTR and conversion rates reviewed
- [ ] Below-average ads flagged for refresh
- [ ] Ad Strength indicators checked
- [ ] Asset performance labels reviewed (PMax: replace "Low" assets)
- [ ] Winning ad themes documented for future creative

## Quality Score
- [ ] Keywords with QS < 5 investigated
- [ ] Low Expected CTR → ad copy improvements planned
- [ ] Low Ad Relevance → keyword-to-ad alignment checked
- [ ] Low Landing Page → web team notified
- [ ] Overall QS trend tracked

## Conversion Health
- [ ] Conversion tracking verified (no sudden drops or spikes)
- [ ] Conversion rate trend reviewed (flag >20% change)
- [ ] Conversion types reviewed (application vs. registration vs. call)
- [ ] Conversion values verified
- [ ] Any tracking issues escalated immediately

## Audiences & Targeting
- [ ] Demographic performance reviewed
- [ ] Geographic performance reviewed
- [ ] Audience segment performance reviewed (PMax)
- [ ] Remarketing list sizes checked (not expired)
- [ ] Bid adjustments by audience reviewed

## Competitive Landscape
- [ ] Auction Insights reviewed (Search)
- [ ] Impression share changes noted
- [ ] New competitors identified
- [ ] Strategy adjustments made if needed

## Landing Pages
- [ ] Landing page uptime verified
- [ ] Mobile experience spot-checked
- [ ] Page speed tested (monthly)
- [ ] Form functionality verified

## Weekly Summary
- [ ] All changes documented in optimization log
- [ ] Key metrics recorded: spend, conversions, CPA, ROAS
- [ ] Top 3 wins noted
- [ ] Top 3 concerns flagged
- [ ] Focus areas set for next week
- [ ] Stakeholder updates sent (if needed)
