# Recruitment Marketing Flex — Coding Standards

## Extends: AIOX Core Standards

### Naming Conventions
- Campaign names: `{channel}-{market}-{jobcategory}-{objective}-{date}` (e.g., `google-nyc-warehouse-acquisition-2026q1`)
- UTM taxonomy: `utm_source={channel}&utm_medium={type}&utm_campaign={campaign_name}&utm_content={variant}`
- Audience names: `{channel}-{segment}-{location}` (e.g., `meta-lookalike-warehouse-nyc`)

### Documentation Standards
- All campaigns must have a brief document before launch
- Performance reports use standardized metric definitions
- A/B tests documented with hypothesis, variants, results, and decision

### Data Standards
- All dates in ISO 8601 format
- Currency in USD with two decimal places
- Percentages to two decimal places
- Geographic markets use metro area names (not zip codes in docs)
