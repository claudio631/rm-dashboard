## 9. Best Practices Applied

<!-- Scout fills this per channel on *new-test -->

### Google Ads
- [ ] One variable changed between control and variant
- [ ] Both ads in same ad group to normalize auction exposure
- [ ] Minimum 30 conversions per variant before reading
- [ ] Ad rotation set to "Do not optimize" during test
- [ ] RSAs: test pinning headline 1 vs unpinned

### Indeed Ads
- [ ] Same job category and location across variants
- [ ] Test title phrasing (hourly rate visible vs hidden)
- [ ] Minimum 50 apply starts per variant
- [ ] Sponsored level identical across variants
- [ ] Run minimum 7 days (avoid day-of-week bias)

### Reddit Ads
- [ ] Same subreddit targeting across variants
- [ ] Free-form ad format preferred over image-only
- [ ] Minimum 50 results per variant
- [ ] Community tone matched in copy (no corporate speak)
- [ ] Check frequency — cap at 3–4× before creative fatigue

### Meta Ads
- [ ] Use Campaign Budget Optimization only with established baselines
- [ ] Audience Overlap check before launching
- [ ] Minimum 50 results per variant
- [ ] Frequency cap set for awareness tests (3×/week)
- [ ] Video: test hook in first 3 seconds as the primary variable

### TikTok Ads
- [ ] Hook test = first 3 seconds only — keep rest identical
- [ ] Sound-on and sound-off versions reviewed
- [ ] Minimum 500 video views per variant
- [ ] Use Spark Ads for organic-looking creative
- [ ] Caption CTA tested separately from creative hook
