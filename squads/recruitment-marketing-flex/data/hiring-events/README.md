# Hiring Events Knowledge Base

> Shared resource for all recruitment-marketing-flex squad agents.
> Add playbooks, post-mortems, checklists, and ideas here.

## Purpose

Hiring events are an in-person funnel acceleration strategy. Workers attend a physical location to complete AI interviews, drug testing, I-9 verification, and get matched to shifts same-day. This folder centralizes all knowledge so any agent can reference it.
