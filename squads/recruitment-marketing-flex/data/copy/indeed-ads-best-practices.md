# Indeed Ads — Copy Best Practices
# Specific to Indeed Sponsored Jobs & Indeed Flex candidate acquisition
# Last Updated: 2026-04-02

---

## How Indeed's Algorithm Ranks Sponsored Jobs

Indeed ranks sponsored listings based on a combination of bid amount AND relevance signals. Copy quality directly affects relevance score and apply-through rate (ATR), which in turn affects cost-per-apply.

| Signal | Copy Impact |
|--------|------------|
| Pay Transparency | Pay in title/description = higher relevance score |
| Job Title Match | Title must match common search queries |
| Apply-Through Rate | Compelling copy → more clicks → better ranking |
| Candidate Experience | Clear requirements reduce unqualified applies |
| Completion Rate | Candidates who finish the application = positive signal |

**Bottom line:** Better copy = lower CPA, not just higher CTR.

## Job Title Rules

### Do
- Use the title candidates actually search for
- Keep it under 60 characters
- Include pay range when character space allows
- Include city/state for local roles
- Use pipe `|` or dash `-` as separators

### Don't
- Add exclamation marks or all-caps
- Use internal job codes or jargon (e.g., "F&B Associate III")
- Stuff keywords (e.g., "Warehouse Worker Forklift Driver Loader")
- Use "Now Hiring:" as a prefix — wastes title space
- Include company name (Indeed shows it separately)

### Title Formulas
```
[Job Title] | $[Pay]/hr | [City, State]
[Job Title] – Flexible Shifts | [City, State]
[Job Title] | [City, State] | Same-Day Pay
```

### Examples
| Weak | Strong |
|------|--------|
| Now Hiring: Server | Event Server \| $21/hr \| Austin, TX |
| Loader/Crew Member Needed! | Loader / Crew \| $16.50/hr \| Orlando, FL |
| Line Cook - Great Pay | Event Line Cook \| $15–$17/hr \| Austin, TX |

## Job Description Structure (Ranked by Apply-Through Rate Impact)

Order content in this sequence for maximum apply rate:

1. **Opening hook** (1–2 sentences) — what the job is + why it's different
2. **Pay rate** — if not in title, within first 3 lines
3. **Key benefits** (bullets) — Same-Day Pay first, then schedule, then insurance
4. **Responsibilities** (bullets) — keep to 4–6 items
5. **Requirements** (bullets) — keep to 3–5 true hard requirements only
6. **Interview/logistics info** — dress code, AI interview note, etc.

### Length
- **Optimal:** 150–400 words
- **Under 100 words:** Too thin — penalized by Indeed algorithm
- **Over 600 words:** Drop-off increases sharply on mobile

## Pay Display Rules

Indeed's research shows listings with pay display receive **30–50% more applies**.

- Always include pay rate (hourly preferred for hourly roles)
- Pay ranges outperform single rates for apply volume (shows flexibility)
- Format: `$16.50/hr` or `$15–$17/hr` — not "competitive pay" or "DOE"
- For event/variable roles: include the range (e.g., `$15–$21/hr depending on event`)

---

## Mobile-First Formatting

Over **75% of Indeed applies** come from mobile devices.

- Use bullet points — no paragraphs over 2 sentences
- Bold section headers
- Lead with the most important info (pay, schedule flexibility)
- Avoid tables — they break on mobile
- Short sentences (under 20 words each)

## Indeed Flex-Specific Copy Rules

| Element | Rule |
|---------|------|
| AI Interview mention | Always include — reduces no-show rate |
| Dress code | Include when mandatory — filters unqualified applies |
| Same-Day Pay | Use exact phrase "Same Day Pay" or "same-day pay" — it's a search term |
| Shift type | "Event-Based", "Flexible Schedule", or "Shifts Vary" — be honest, improves retention |
| "Indeed Flex is hiring" | Use in first line of description — reinforces brand and staffing model |

## CTAs That Work on Indeed

| CTA | Context |
|-----|---------|
| Apply Now | Default — always works |
| See Open Shifts | When multiple roles or shift types available |
| Apply in 2 Minutes | When application is short (AI interview flow) |
| Start This Week | When immediate hiring is true |
| Join the Team | Warmer tone for hospitality/events roles |

## Bilingual (Spanish) Best Practices on Indeed

- Post a **separate Spanish listing** — do not just paste a translation into the English listing
- Use the same job title in Spanish for discoverability (`Cargador / Crew | Orlando, FL`)
- Spanish listings reach a separate audience segment — not the same candidates
- "Pago el mismo día" is a high-search Spanish term — include it
- Match the hashtag/job code between EN and ES versions for tracking

## Common Copy Mistakes That Hurt Performance

| Mistake | Impact |
|---------|--------|
| No pay rate | -30 to -50% applies |
| Requirements list > 7 items | High drop-off, unqualified applies stay longer |
| Dress code only at the end | Candidates apply without reading it — no-show waste |
| Generic job title | Low CTR, poor keyword match |
| Repeating the same benefit 3 ways | Reads as filler, reduces trust |
| "Must be able to pass background check" in requirements | Scares off candidates unnecessarily — add it in logistics section instead |
