# Google Ads Ad Copywriting Playbook

> Source: [Google Business Resource Hub](https://business.google.com/us/resources/)

## Ad Format Specifications

### Responsive Search Ads (RSA)
| Element | Limit | Count |
|---------|-------|-------|
| Headlines | 30 characters each | 3-15 headlines |
| Descriptions | 90 characters each | 2-4 descriptions |
| Display Path | 15 characters each | 2 paths |
| Final URL | Full URL | 1 |

### Responsive Display Ads
| Element | Specification |
|---------|--------------|
| Short headline | 25 characters |
| Long headline | 90 characters |
| Description | 90 characters |
| Business name | 25 characters |
| Images | Landscape (1200x628), Square (1200x1200), Logo (1200x1200) |

## 9 Core Copywriting Principles

### 1. Know Your Audience
Research what candidates value most:
- Pay rate and benefits
- Schedule flexibility
- Location/commute time
- Growth opportunities
- Work environment

**Recruitment tip:** Survey existing workers to learn what attracted them to apply.

### 2. Highlight Unique Value Proposition
Differentiate from competing employers:
- "Same-day pay available"
- "Choose your own schedule"
- "No experience needed — paid training"
- "Weekly pay starting at $18/hr"

### 3. Create Urgency
Use temporal cues to drive immediate action:
- "Hiring this week"
- "Limited spots — apply now"
- "Start as soon as tomorrow"
- "Spring hiring event — March only"

### 4. Include Clear Calls-to-Action
Strong CTAs for recruitment:
| CTA | When to Use |
|-----|-------------|
| Apply Now | Direct application flow |
| Sign Up Today | Registration-based funnels |
| See Open Shifts | Browse-first experience |
| Call to Apply | Phone-based application |
| Get Started | Low-commitment first step |
| Learn More | Awareness/consideration stage |

### 5. Use Keywords in Headlines
Mirror candidate search intent:
- Search: "warehouse jobs Houston" → Headline: "Warehouse Jobs in Houston"
- Search: "temp work near me" → Headline: "Temp Work — Start This Week"

### 6. Be Specific and Relevant
Write unique copy per ad group — never reuse generic ads:
- Bad: "Great Jobs Available — Apply Now"
- Good: "Forklift Operator — $20/hr — Dallas | Apply in 2 Minutes"

### 7. Keep Language Direct and Clear
Prioritize clarity over cleverness:
- Bad: "Your next career adventure awaits!"
- Good: "Warehouse Jobs — $18/hr — Flexible Shifts"

### 8. Maintain Professional Standards
- Correct grammar and spelling
- No ALL CAPS (violates Google policy)
- No excessive punctuation (!!!)
- No misleading claims

### 9. Test and Iterate
- Change one element at a time
- Allow sufficient data before judging (1-2 weeks minimum)
- Use Ad Strength indicator as a guide
- Create 3+ headline variations per theme
- Test emotional vs. rational appeals

## Ad Copy Formulas for Recruitment

### Formula 1: Job + Pay + Location
```
Headline 1: {Job Title} — ${Pay}/hr
Headline 2: {City} | Hiring Now
Headline 3: Apply in Under 2 Minutes
Description: Looking for {job title} work in {city}? We're hiring immediately. Flexible schedules, weekly pay, and no experience needed. Apply today.
```

### Formula 2: Benefit-Led
```
Headline 1: Choose Your Own Schedule
Headline 2: {Job Title} Jobs — {City}
Headline 3: Same-Day Pay Available
Description: Work when you want with flexible {job type} shifts. Earn ${pay}/hr with same-day pay. No long-term commitment required. Sign up free.
```

### Formula 3: Urgency-Driven
```
Headline 1: We're Hiring This Week
Headline 2: {Job Title} — Start ASAP
Headline 3: ${Pay}/hr + Benefits
Description: {Number} open {job title} positions in {city}. Start as soon as tomorrow. Paid training, flexible hours, and weekly pay. Apply now — spots filling fast.
```

### Formula 4: Question-Based
```
Headline 1: Looking for {Job Type} Work?
Headline 2: ${Pay}/hr in {City}
Headline 3: No Experience Needed
Description: Find flexible {job type} jobs near you. We're hiring in {city} with shifts that fit your schedule. Apply in minutes — start earning this week.
```

## AI-Assisted Copywriting

### Using Google's AI Features
1. **Responsive Search Ads:** Provide 15 headlines + 4 descriptions; AI tests combinations
2. **Ad Strength indicator:** Aim for "Good" or "Excellent" rating
3. **Asset suggestions:** Accept or modify Google's recommended headlines
4. **Keyword insertion:** Use `{KeyWord:Default}` for dynamic matching

### Human + AI Balance
- **Human provides:** Brand voice, unique value props, compliance requirements
- **AI optimizes:** Combinations, testing at scale, performance-based serving
- **Best practice:** Write 60% of assets manually, let AI suggest 40%

### AI Copywriting Tips
1. Include at least one keyword in each headline and description
2. Use keyword insertion for dynamic relevance
3. Leverage sitelinks, callouts, and image extensions
4. Add lead form extensions matching CTAs to search intent
5. Test variations using ad experiments feature

## Ad Extensions for Recruitment

| Extension | Example | Impact |
|-----------|---------|--------|
| Sitelinks | "Browse Warehouse Jobs", "See Pay Rates", "How to Apply" | +10-15% CTR |
| Callouts | "Same-Day Pay", "Flexible Hours", "No Experience Needed" | Highlights benefits |
| Structured Snippets | "Roles: Warehouse, Delivery, Caregiver, Retail" | Shows breadth |
| Location | Shows nearest hiring location | Drives local relevance |
| Call | Click-to-call for phone applications | Mobile conversion |
| Lead Form | In-ad application form | Reduces friction |
| Image | Workplace photos | Visual engagement |
