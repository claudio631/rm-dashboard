# Performance Max Playbook

> Source: [Google Business Resource Hub](https://business.google.com/us/resources/)

## Overview

Performance Max is Google's AI-powered campaign type that serves ads across **all Google channels** (Search, Display, YouTube, Gmail, Maps, Discover) from a single campaign. It uses Smart Bidding to optimize toward your business goals in real-time.

**Key stat:** Advertisers using Performance Max report an average **+18% incremental conversions** at similar cost per action.

---

## When to Use Performance Max

| Scenario | Recommendation |
|----------|---------------|
| Want cross-channel reach from one campaign | Use PMax |
| Have strong creative assets + first-party data | Use PMax |
| Need precise keyword control | Use Search instead |
| Complementing existing Search campaigns | Use PMax alongside |
| Limited creative assets | Build assets first, then PMax |
| Testing a new market | Start with Search, add PMax after baseline |

## Setup Checklist

### 1. Define Conversion Goals
- [ ] Set primary conversion action (e.g., application submitted)
- [ ] Set secondary conversions (e.g., registration started, phone call)
- [ ] Assign conversion values (application = $X, registration = $Y)
- [ ] Verify Google Tag is installed sitewide
- [ ] Enable enhanced conversions for accuracy

### 2. Build Asset Groups
- [ ] Create 15 headlines (30 chars each)
- [ ] Create 5 long headlines (90 chars each)
- [ ] Create 5 descriptions (90 chars each)
- [ ] Upload 3+ landscape images (1200x628)
- [ ] Upload 3+ square images (1200x1200)
- [ ] Upload 1+ portrait images (960x1200)
- [ ] Upload or create 1+ videos (if none, Google auto-generates)
- [ ] Add logo (1200x1200 + 1200x300)
- [ ] Set display URL paths
- [ ] Set final URL(s)

### 3. Configure Audience Signals
- [ ] Upload Customer Match lists (existing candidates/workers)
- [ ] Create custom segments from keywords and URLs
- [ ] Add remarketing lists (site visitors, app users)
- [ ] Define demographic targets
- [ ] Set interest-based audiences

### 4. Set Budget and Bidding
- [ ] Choose bidding strategy: Maximize Conversions or Maximize Conversion Value
- [ ] Set optional Target CPA or Target ROAS
- [ ] Set daily budget (recommend: ≥10x your target CPA)
- [ ] Plan for 6-week ramp-up period

### 5. Launch and Monitor
- [ ] Review campaign settings
- [ ] Activate campaign
- [ ] Check asset group strength ("Excellent" target)
- [ ] Monitor daily for first 2 weeks (do not change settings)
- [ ] Review Insights page after 2 weeks

## Asset Optimization

### Asset Requirements (Minimum → Recommended)

| Asset Type | Minimum | Recommended | Maximum |
|------------|---------|-------------|---------|
| Headlines (30 char) | 5 | 11-15 | 15 |
| Long headlines (90 char) | 1 | 3-5 | 5 |
| Descriptions (90 char) | 2 | 4-5 | 5 |
| Landscape images | 1 | 5+ | 15 |
| Square images | 1 | 5+ | 15 |
| Portrait images | 0 | 2+ | 15 |
| Videos (YouTube) | 0 | 3+ | 5 |
| Logos (square) | 1 | 2 | 5 |
| Logos (landscape) | 0 | 1 | 5 |

### Asset Best Practices
1. **Diversity:** Vary messaging themes (pay, flexibility, location, speed)
2. **Specificity:** Include numbers, locations, and concrete benefits
3. **Headlines:** Mix branded + benefit + CTA headlines
4. **Images:** Use real workplace photos, diverse talent, action shots
5. **Videos:** 15-30s employer brand or "day in the life" content
6. **Refresh:** Update underperforming assets monthly

### Asset Performance Labels
| Label | Meaning | Action |
|-------|---------|--------|
| Best | Top performer | Keep, create similar |
| Good | Solid performer | Keep |
| Low | Below average | Replace within 30 days |
| Pending | Insufficient data | Wait for more impressions |

## Audience Signals

Audience signals **guide** (not restrict) Google's AI. The better your signals, the faster PMax ramps up.

### Signal Types for Recruitment

| Signal Type | Example | Purpose |
|-------------|---------|---------|
| Customer Match | Previous applicant emails | Find similar candidates |
| Website Visitors | Career page visitors | Remarketing |
| Custom Segments (Keywords) | "warehouse jobs hiring" | Intent matching |
| Custom Segments (URLs) | Competitor career pages | Competitor targeting |
| Interests & Demographics | Age 18-45, job seekers | Broad qualification |
| Life Events | Recently moved, new graduates | Contextual targeting |

### Signal Priority
1. Customer Match (highest signal quality)
2. Website remarketing lists
3. Custom keyword segments
4. Custom URL segments
5. Interest/demographic audiences

## Optimization Tips

### First 6 Weeks (Ramp-Up)
- Do NOT change budget, bidding, or campaign settings
- Monitor but don't intervene
- Check asset performance labels (replace "Low" assets only)
- Review search term insights for negative keyword opportunities

### Ongoing Optimization
1. **Review Insights page weekly** — top audiences, search themes, asset performance
2. **Add negative keywords** at account level for irrelevant search themes
3. **Refresh creative assets** monthly — swap underperformers
4. **Expand audience signals** based on conversion data
5. **Combine with Search campaigns** — PMax fills gaps Search doesn't cover

### Performance Max + Search Strategy
```
Search Campaign (keyword-targeted)
  → Captures high-intent, specific job searches
  → You control exact keywords and bids

Performance Max Campaign
  → Extends reach across all channels
  → Finds candidates Search misses
  → Uses AI to optimize placement and creative

Together: 18%+ incremental conversions vs Search alone
```

## Measurement and Reporting

### Key Metrics
| Metric | What It Tells You |
|--------|------------------|
| Conversions | Total applications/registrations |
| Conversion Value | Revenue or value of conversions |
| Cost per Conversion | Average cost per application |
| ROAS | Return on ad spend |
| Asset Performance | Which headlines/images work best |
| Audience Insights | Which segments convert best |
| Search Term Insights | What candidates searched for |

### Reporting Cadence
| Frequency | Review |
|-----------|--------|
| Daily (first 2 weeks) | Spend pacing, any errors |
| Weekly | Conversions, CPA, asset performance |
| Monthly | Full performance review, asset refresh |
| Quarterly | Strategy review, budget reallocation |
