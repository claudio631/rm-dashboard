# Lead Generation Guide for Google Ads

> Source: [Google Business Resource Hub](https://business.google.com/us/resources/)

## Lead Generation Strategies

### 1. Google Search Ads (Primary)
- **Model:** Pay-per-click
- **Best for:** High-intent candidates actively searching
- **Key features:** Keyword targeting, geographic control, bid management
- **Lead type:** Form submissions, phone calls

### 2. Local Services Ads (LSA)
- **Model:** Pay-per-lead (only pay for qualified leads)
- **Best for:** Location-specific hiring, service-area businesses
- **Key features:**
  - Appear at the very top of Search results
  - Leads via phone, text, email, or booking
  - No keyword research required
  - No website required
- **Lead type:** Phone calls, messages, bookings
- **Trust badges:** Google Guarantee, Google Screened, License Verified

### 3. Performance Max
- **Model:** Smart Bidding (CPA or ROAS)
- **Best for:** Cross-channel lead generation at scale
- **Key features:** AI optimization across all Google channels
- **Lead type:** Website conversions, phone calls

### 4. Display Remarketing
- **Model:** CPC or CPA bidding
- **Best for:** Re-engaging candidates who visited but didn't apply
- **Key features:** Visual ads on 2M+ sites
- **Lead type:** Return visit conversions

### 5. YouTube Ads
- **Model:** CPV, CPM, or CPA
- **Best for:** Building awareness → consideration → application
- **Key features:** Video storytelling, audience targeting
- **Lead type:** Post-view conversions, direct clicks

```
AWARENESS

```

## Lead Form Extensions

### Setup
Add lead forms directly within Google Ads:
1. Create at campaign or ad group level
2. Define form fields (name, email, phone, zip code)
3. Set CTA text ("Apply Now", "Sign Up", "Get Started")
4. Add form headline and description
5. Set thank-you message with next steps

### Benefits
- **Zero page load** — form opens within the ad
- **Pre-filled data** — Google auto-fills from user's account
- **Higher conversion rates** — reduces friction
- **Works across Search, YouTube, Display, Discovery**

### Best Practices
1. Keep form fields to minimum (3-5 fields max)
2. Match CTA to search intent
3. Set clear expectations in form description
4. Use webhook or CRM integration for immediate follow-up
5. Download leads daily if not using integration

## Geographic Targeting for Lead Gen

### Location Targeting Options
| Level | Example | Use Case |
|-------|---------|----------|
| Country | United States | National campaigns |
| State | Texas | State-level hiring pushes |
| City/Metro | Dallas-Fort Worth | Market-specific campaigns |
| Radius | 25 miles from office | Commute-distance targeting |
| Zip Code | 75201, 75202, 75203 | Hyper-local targeting |

### Geo-Targeting Best Practices
1. Target only markets with active open positions
2. Use radius targeting around job sites (15-30 mile radius)
3. Adjust bids by location based on application rates
4. Test individual cities before expanding regionally
5. Exclude areas without hiring needs
6. Monitor location reports for wasted spend

## Lead Quality Optimization

### Improving Lead Quality
1. **Qualify in the ad** — mention requirements (e.g., "Must have valid driver's license")
2. **Set conversion values** — assign higher value to completed applications vs. partial registrations
3. **Use negative keywords** — exclude irrelevant searches
4. **Optimize for downstream actions** — import offline conversion data back to Google Ads
5. **Score leads** — track which campaigns/keywords produce candidates who complete onboarding

### Lead Scoring Framework
| Signal | Points | Rationale |
|--------|--------|-----------|
| Application completed | +50 | Primary conversion |
| Phone call >60 seconds | +30 | Engaged candidate |
| Registration started | +10 | Top-of-funnel interest |
| Multiple page views | +5 | Research behavior |
| Return visit | +15 | Ongoing interest |
| From remarketing | +10 | Known audience |
