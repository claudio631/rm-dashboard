# Optimization Playbook

## Week 1 (Days 1-7): Learning Phase
- **DO:** Let campaigns spend. Resist making changes.
- **MONITOR:** Impression volume, CTR, CPC by ad group
- **ACTION:** Only pause if CPC > $5 or obvious keyword mismatches in Search Terms
- **CL POSTS:** Monitor response rate; test different posting times if low

## Week 2 (Days 8-14): First Optimizations
- **Google:** Review Search Terms report — add 15+ negatives. Pause keywords with spend >$20 and zero conversions
- **Meta:** Kill ad variants with CTR < 0.5%. Increase spend on winner
- **Reddit:** Evaluate community targeting vs keyword targeting — shift budget to winner
- **Craigslist:** A/B test title format (pay-first vs role-first)

## Week 3 (Days 15-21): Scale Winners
- **Google:** Switch to Target CPA ($18) if 30+ conversions reached. Expand exact match keywords from Search Terms
- **Meta:** Create Lookalike Audience from LP converters if volume sufficient
- **Budget Shift:** Move 20% budget from worst channel to best channel
- **Check:** Are we on pace for 100 RSVPs? If behind, consider budget increase

## Week 4 (Days 22-30): Close or Extend
- **Full audit:** CPA by channel, quality of RSVPs (show-up rate if available)
- **Decision:** If target hit → close. If under 70 RSVPs → extend + increase budget
- **Document:** Final performance report for future Levy campaigns
